# Error Code Registry

## 에러 응답 형식

```json
{
  "success": false,
  "data": null,
  "error": {
    "code": "M001",
    "message": "이미 등록된 이메일입니다"
  }
}
```

---

## 공통 에러 (Common: C001 ~ C010)

| 코드 | HTTP 상태 | 메시지 | 설명 |
|------|-----------|--------|------|
| C001 | 400 | 잘못된 요청입니다 | 요청 파라미터 유효성 검증 실패 |
| C002 | 401 | 인증이 필요합니다 | 인증 토큰 누락 또는 만료 |
| C003 | 403 | 접근 권한이 없습니다 | 인가 실패 (권한 부족) |
| C004 | 404 | 요청한 리소스를 찾을 수 없습니다 | 존재하지 않는 리소스 접근 |
| C005 | 405 | 허용되지 않는 HTTP 메서드입니다 | 지원하지 않는 HTTP Method |
| C006 | 409 | 리소스 충돌이 발생했습니다 | 동시성 충돌 (Optimistic Lock 등) |
| C007 | 422 | 처리할 수 없는 요청입니다 | 비즈니스 규칙 위반 (일반) |
| C008 | 429 | 요청이 너무 많습니다 | Rate Limit 초과 |
| C009 | 500 | 서버 내부 오류가 발생했습니다 | 예상하지 못한 서버 오류 |
| C010 | 503 | 서비스를 일시적으로 사용할 수 없습니다 | 외부 서비스 장애 또는 점검 |

## Member 에러 (M001 ~ M010)

| 코드 | HTTP 상태 | 메시지 | 설명 |
|------|-----------|--------|------|
| M001 | 409 | 이미 등록된 이메일입니다 | 회원가입 시 이메일 중복 |
| M002 | 401 | 비밀번호가 일치하지 않습니다 | 로그인 또는 비밀번호 변경 시 불일치 |
| M003 | 404 | 회원을 찾을 수 없습니다 | 존재하지 않는 회원 ID |
| M004 | 401 | 유효하지 않은 토큰입니다 | JWT 토큰 검증 실패 (서명 불일치, 변조) |
| M005 | 401 | 만료된 토큰입니다 | JWT 토큰 만료 |
| M006 | 401 | 이미 로그아웃된 토큰입니다 | 블랙리스트에 등록된 토큰 사용 |
| M007 | 403 | 등급이 불충분합니다 | 등급 제한 기능 접근 시 (VIP 전용 등) |
| M008 | 400 | 유효하지 않은 이메일 형식입니다 | 이메일 형식 검증 실패 |
| M009 | 400 | 비밀번호 형식이 올바르지 않습니다 | 비밀번호 정책 미충족 (최소 8자, 대소문자+숫자+특수문자) |
| M010 | 404 | 배송지를 찾을 수 없습니다 | 존재하지 않는 배송지 ID |

## Product 에러 (P001 ~ P010)

| 코드 | HTTP 상태 | 메시지 | 설명 |
|------|-----------|--------|------|
| P001 | 422 | 상품 상태를 변경할 수 없습니다 | 허용되지 않는 상태 전이 (예: SOLD_OUT → DRAFT) |
| P002 | 409 | 이미 존재하는 SKU입니다 | SKU 중복 등록 |
| P003 | 404 | 카테고리를 찾을 수 없습니다 | 존재하지 않는 카테고리 ID |
| P004 | 404 | 상품을 찾을 수 없습니다 | 존재하지 않는 상품 ID |
| P005 | 400 | 상품 가격은 0보다 커야 합니다 | 가격 유효성 검증 실패 |
| P006 | 400 | 재고 수량은 0 이상이어야 합니다 | 재고 수량 음수 방지 |
| P007 | 422 | 판매 중인 상품은 삭제할 수 없습니다 | ACTIVE 상태 상품 삭제 시도 |
| P008 | 400 | 상품 이름은 필수입니다 | 상품명 누락 |
| P009 | 422 | 하위 카테고리가 존재하여 삭제할 수 없습니다 | 자식 카테고리 존재 시 부모 삭제 시도 |
| P010 | 409 | 이미 존재하는 카테고리 이름입니다 | 같은 레벨에서 카테고리명 중복 |

## Order 에러 (O001 ~ O010)

| 코드 | HTTP 상태 | 메시지 | 설명 |
|------|-----------|--------|------|
| O001 | 422 | 재고가 부족합니다 | 주문 수량이 가용 재고 초과 |
| O002 | 404 | 주문을 찾을 수 없습니다 | 존재하지 않는 주문 ID |
| O003 | 422 | 주문 상태를 변경할 수 없습니다 | 허용되지 않는 상태 전이 (예: CONFIRMED → CANCEL_REQUESTED) |
| O004 | 422 | 품절된 상품이 포함되어 있습니다 | 주문 생성 시 품절 상품 포함 |
| O005 | 400 | 주문 항목이 비어있습니다 | 주문 아이템 없이 주문 생성 시도 |
| O006 | 422 | 이미 취소된 주문입니다 | 취소된 주문에 대해 재취소 시도 |
| O007 | 422 | 배송 완료 후에만 구매 확정이 가능합니다 | DELIVERED가 아닌 상태에서 구매 확정 시도 |
| O008 | 422 | 배송 완료 후에만 반품이 가능합니다 | DELIVERED가 아닌 상태에서 반품 시도 |
| O009 | 400 | 주문 금액이 일치하지 않습니다 | 클라이언트 전달 금액과 서버 계산 금액 불일치 |
| O010 | 403 | 본인의 주문만 조회할 수 있습니다 | 다른 회원의 주문 접근 시도 |

## Payment 에러 (PAY001 ~ PAY010)

| 코드 | HTTP 상태 | 메시지 | 설명 |
|------|-----------|--------|------|
| PAY001 | 422 | PG 결제 승인에 실패했습니다 | PG사 API 승인 거절 (카드 한도, 잔액 부족 등) |
| PAY002 | 422 | 이미 취소된 결제입니다 | CANCELLED 상태 결제에 재취소 시도 |
| PAY003 | 422 | 환불 금액이 결제 금액을 초과합니다 | 부분 취소 금액 합계가 원 결제 금액 초과 |
| PAY004 | 404 | 결제 정보를 찾을 수 없습니다 | 존재하지 않는 결제 ID |
| PAY005 | 422 | 결제 금액이 일치하지 않습니다 | 승인 요청 금액과 준비 금액 불일치 |
| PAY006 | 502 | PG사 연동 오류가 발생했습니다 | PG API 호출 실패 (타임아웃, 네트워크 오류) |
| PAY007 | 422 | 이미 승인된 결제입니다 | APPROVED 상태 결제에 재승인 시도 |
| PAY008 | 422 | 결제 준비 상태에서만 승인이 가능합니다 | READY가 아닌 상태에서 승인 시도 |
| PAY009 | 422 | 지원하지 않는 결제 수단입니다 | 허용되지 않은 결제 방식 |
| PAY010 | 422 | 결제 유효 시간이 초과되었습니다 | 결제 준비 후 일정 시간 내 미승인 |
