# AI Wiki Roadmap

## Now (MVP / 즉시 해결)
1. 상태 모델 분리
   - 문서 상태와 AI 처리 상태 분리
   - API 응답 상태값 일치
2. 버전 관리 범위 확정
   - revision에 포함되는 필드 정의
3. 태그/요약 스키마 정합성
   - 태그는 1:N, 요약은 1:1 유지
4. 삭제 정책 및 기본 조회 규칙
   - soft delete 필터링 기준
5. AI 파이프라인 기본 규칙
   - 이벤트 순서, 재시도, 중복 처리 정책
6. SSE 스펙 확정
   - 이벤트 포맷/heartbeat/재연결 정책

## Next (1~2 마일스톤)
1. 검색 고도화
   - 풀텍스트 vs 외부 검색엔진 결정
   - 통합 검색 랭킹 정책
2. 임베딩 관리 전략
   - 모델 버전/차원 관리 및 재인덱싱
3. 보안/운영 기본기
   - refresh 토큰 회전/저장 정책
   - 로그 보관 및 비용 모니터링

## Later (확장 로드맵)
1. 권한/공유 모델
   - 팀/조직/ACL/공유 링크
2. 협업 기능
   - 코멘트, 변경 diff, 알림/웹훅
3. 멀티 에이전트 확장
   - 정책 기반 에이전트 플러그인화
