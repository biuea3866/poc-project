# 엑셀 다운로드
## 요구 사항
1. JVM을 이용하여 엑셀을 다운로드하는 기능을 제공한다.
2. 엑셀 다운로드를 위한 관련 라이브러리를 조사하고 결과를 md 파일로 한글로 작성한다.
3. 다음의 플로우로 엑셀 다운로드를 제공한다.
    * 1. 유저의 주문 내역을 엑셀로 다운로드한다.
    * 2. 주문 내역은 상품, 쿠폰, 주문 정보로 구성되어있다.
    * 3. 주문 내역의 정보는 사용자가 원하는 열만을 동적으로 구성하여 다운 받을 수 있다.
      * e.g 1) 상품의 가격은 제외하고 주문 내역을 다운받는다.
      * e.g 2) 쿠폰의 할인율은 제거하고 주문 내역을 다운받는다.
    * 4. 시트는 주문 내역 시트, 상품 시트 2가지 시트로 구성되어 있다.
    * 5. a ~ c까지의 공통 로직에 대한 결과를 1개는 api로 동기적으로 응답해서 브라우저로 다운로드해주고, 1개는 비동기적으로 처리하고 유저 이메일로 발송한다.
    * 6. 현재 디렉토리에 예제 플로우를 작성한다.
    * 7. 주석, 테스트 코드 전부 한글로 작성한다.
