# 의류 관리 가이드

## 코튼 (면)

- 미지근한 물(30°C 이하)에 단독 세탁. 색 옷과 분리.
- 청바지는 뒤집어서 세탁하면 색빠짐을 줄일 수 있습니다.
- 건조기 사용 시 수축할 수 있으므로 그늘에서 자연 건조 권장.

## 울 · 캐시미어 (니트)

- 드라이클리닝 권장. 가정 세탁 시 30°C 미만 손빨래 + 중성 세제.
- 비틀어 짜지 말고 수건으로 흡습 후 평평하게 눕혀 건조.
- 보관 시 옷걸이 대신 접어서 보관 (어깨 늘어남 방지).

## 다운 (구스다운 · 덕다운 패딩)

- 가정 세탁 가능. 30°C 이하 + 중성 세제 + 약한 회전.
- 헹굼을 두 번 이상 충분히 (세제 잔류 시 보온성 저하).
- 완전 건조까지 24시간 이상 — 다 마르기 전에 입으면 곰팡이 발생.
- 시즌 종료 후 압축 보관 금지. 통풍이 잘 되는 곳에 부피 유지.

## 가죽 · 스웨이드

- 물세탁 금지. 전용 클리너로 가볍게 닦아내는 정도.
- 직사광선·습기 회피. 옷장 보관 시 방습제 필수.
