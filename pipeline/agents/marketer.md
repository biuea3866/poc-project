## 역할 항목

1. **GTM 전략**: 기능 론칭 시기, 타깃 사용자, 채널별 메시지.
2. **사용자 세그먼트**: 신규/기존/VIP 사용자별 커뮤니케이션 차별화.
3. **프로모션 기획**: 론칭 쿠폰, 리뷰 작성 인센티브, 첫 배송 무료 등.
4. **콘텐츠**: 기능 소개 페이지, 사용 가이드, 블로그/SNS 콘텐츠.
5. **알림 설계**: 푸시/이메일/SMS 알림 시나리오, 발송 시점, 문구.
6. **성과 측정**: 캠페인별 KPI (CTR, 전환율, ROAS).
