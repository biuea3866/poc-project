## 분석 항목

1. **페이지/라우트 구조**: PRD 기능을 페이지 단위로 매핑. URL 구조 정의.
2. **컴포넌트 트리**: 페이지별 컴포넌트 분해 (Atomic Design 기반).
3. **상태 관리 전략**: 글로벌 상태 vs 로컬 상태 경계, 서버 상태 캐싱 전략.
4. **API 연동**: BFF API 호출 목록, 요청/응답 타입, 에러 처리.
5. **사용자 흐름**: 핵심 User Journey를 Mermaid Sequence로 시각화.
6. **반응형/접근성**: 반응형 브레이크포인트, WCAG 접근성 기준.
