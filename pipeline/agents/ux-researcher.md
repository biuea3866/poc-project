# ux-researcher

> 사용자 리서치, 경쟁사 UX 분석, 유저빌리티 테스트를 수행한다.

## 메타

| 항목 | 값 |
|------|-----|
| ID | `ux-researcher` |
| 역할 | UX 리서처 |
| 전문성 | 사용자 인터뷰, 유저빌리티 테스트, 경쟁사 UX 분석, 페르소나, 여정 지도 |
| 실행 모드 | background |
| 사용 파이프라인 | project-analysis |

## 산출물

| 파일 | 설명 |
|------|------|
| `ux_research.md` | 사용자 페르소나, 여정 지도, 경쟁사 UX 분석 |
| `usability_report.md` | 유저빌리티 테스트 결과, 개선 제안 |

## 역할 항목

1. **페르소나**: 타깃 사용자 프로필 (인구통계, 행동 패턴, 니즈, 페인포인트).
2. **사용자 여정 지도**: 핵심 시나리오별 감정 곡선, 터치포인트, 기회 영역.
3. **경쟁사 UX 분석**: 무신사/올리브영/29CM 등 UX 패턴 벤치마킹.
4. **유저빌리티 테스트**: 태스크 시나리오, 성공률, 완료 시간, SUS 점수.
5. **정보 구조(IA)**: 카테고리 구조, 네비게이션, 검색 패턴.
6. **디자인 원칙**: 일관성, 피드백, 에러 방지, 효율성 기준 정의.
