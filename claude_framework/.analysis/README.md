# .analysis — 분석 파이프라인

각 서브디렉토리는 하나의 파이프라인이며, `PIPELINE.md`가 진입 문서이다.

## 파이프라인 목록

| 디렉토리 | 용도 |
|---|---|
| `prd/` | PRD 분석 (질의사항, 애매한 점, 경쟁사 리서치) |
| `project-analysis/` | 프로젝트 분석 (TDD 전략, 티켓 분해, 상세 설계) |

## 사용법

1. 작업 성격에 맞는 파이프라인 선택
2. 해당 `PIPELINE.md`를 읽고 지시에 따라 단계 수행
3. 산출물은 같은 디렉토리에 날짜/티켓ID 접두사로 누적 (`2026-04-18-GRT-1234.md` 형식)
